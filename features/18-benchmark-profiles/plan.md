# Benchmark Profiles

Save and load named configurations of servers + domains.

## Changes
- `dns_benchmark.py`: "Profiles" button opens dialog with saved profiles list
- Save current server selection + domain list as named profile to ~/.latensee/profiles.json
- Load profile: restores server checkboxes + domain text box
- Built-in presets: "Gaming" (low-latency focused), "Privacy" (Quad9/AdGuard/NextDNS), "All"
- Profile manager dialog: create, rename, delete
